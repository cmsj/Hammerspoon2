    Function bindHotkey

    • bindHotkey(
          mods: string[],
          key: string,
          callbackPressed: (() => boolean | void) | null,
          callbackReleased: (() => boolean | void) | null,
      ): HSEventTapHotkey | null

      Bind a keyboard shortcut using an event tap. Unlike hs.hotkey.bind(), this supports the fn modifier and left/right modifier key distinction (e.g. leftCmd, rightAlt). The hotkey is active immediately. By default the matching key events are consumed (suppressed). To let the event pass through to other applications, return false from the callback. Returning true or nothing consumes the event as usual. It's important to note that this a much heavier-weight tool than hs.hotkey - every single key you press will be examined by Hammerspoon to see if it matches one of the EventTap hotkeys (where hs.hotkey relies on macOS to efficiently deliver only matching keypresses). Please consider this when choosing to use hs.eventtap for hotkeys.       • mods: string[]

      • key: string

        The key name or character (e.g., "a", "space", "f1")

      • callbackPressed: (() => boolean | void) | null

        Called when the key combination is pressed. Return false to pass the event through, or return true/nothing to consume it. Pass null for no callback.

      • callbackReleased: (() => boolean | void) | null
