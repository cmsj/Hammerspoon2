HSNetworkReachabilityModule

Type

Provides network reachability monitoring using the Network framework.

Reachability objects observe whether the system has network connectivity. Create one with a factory method, register a callback with setCallback(), then start monitoring with start().

The flags property exposes the numeric bitmask constants for interpreting status().

Properties

typeName

string
Always `"HSNetworkReachabilityModule"`. Identifies this object as the reachability sub-module.

flags

{[key: string]: number}
A table of bitmask constants for interpreting the value returned by `HSNetworkReachability.status()`. Keys: `transientConnection`, `reachable`, `connectionRequired`, `connectionOnTraffic`, `interventionRequired`, `connectionOnDemand`, `isLocalAddress`, `isDirect`.
